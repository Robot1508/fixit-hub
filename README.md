# FixIt Hub

FixIt Hub is a smart civic issue reporting platform that allows citizens to report problems like potholes, garbage overflow, broken streetlights, and water leakage.

## Features
- Report issues with image
- GPS location detection
- AI-based issue categorization
- Real-time complaint tracking

## Tech Stack
Frontend: React  
Backend: AWS (bedrock,lambda)
Cloud: AWS

## Problem
Citizens struggle to report civic issues and authorities lack efficient tracking systems.

## Solution
FixIt Hub provides a centralized platform for quick reporting and efficient resolution.
